# Overlap module

```
circuitfp.Overlap
```

The `Overlap` class evaluates the overlap bwteen two BCS phase states.


# Mathematical background

The overlap between two BCS phase states is given by

\[
W(\phi_1,\phi_2)
=
\exp
\left[
n
\left(
\frac{i}{2}\Delta\phi
-
\frac{\pi\Delta}{16B}
(\Delta\phi)^2
\right)
\right]
\]

where \( \varphi = \phi_2 - \phi_1 \) is the phase difference.
